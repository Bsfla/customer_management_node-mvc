# 고객 관리 프로그램
> 심심해서 만든 토이 프로젝트  

* 개발 일자 : 2021년 06월 12일

### API

1. 고객 정보 조회
   -  Input 창에 입력된 search 데이터로 검색
2. 고객 사이트 정보 조회
3. 고객 사이트 정보 수정
4. 고객 사이트 정보 삭제
